# md2pdf

Markdown 파일을 한글 폰트 지원 PDF로 변환합니다.

## 기능

- Markdown → PDF 변환
- 한글 폰트 완벽 지원 (맑은 고딕, Apple Gothic)
- 깔끔한 스타일링 (A4, 헤딩 색상, 코드 배경)
- 파일명만 입력해도 프로젝트 전체 검색

## 설치

```bash
mkdir -p .claude/skills && curl -sL https://github.com/daht-mad/md2pdf/archive/refs/heads/master.tar.gz | tar -xz -C /tmp && mv /tmp/md2pdf-master .claude/skills/md2pdf
```

## 필요 환경

- Node.js 18+
- `npm install md-to-pdf`
